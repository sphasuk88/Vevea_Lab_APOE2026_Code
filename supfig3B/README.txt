# supfig3B## DescriptionThis folder contains an R script used to generate the Pearson sample-correlation heatmap for Supplementary Figure 3B from a proteomics CSV file.----------------------------------------------------------------## Folder structure```textsupfig3B/├── input/│   └── demo_supfig3B.csv├── output/└── R_code/    └── supfig3B.R----------------------------------------------------------------**R version:**This script was tested with: R version 4.5.2----------------------------------------------------------------**How to run the demo:**1. Open RStudio.2. Open the script:3. R_code/supfig3B.R4. Run the script.5. When prompted, choose the file:6. input/demo_supfig3B.csv7. The script will automatically:8. detect the project folder9. create the output folder if needed10. generate the heatmap PDF11. save the output file in the output folder----------------------------------------------------------------**Expected output:**The script produces a Pearson sample-correlation heatmap.> demo_supfig3B.pdfThis file is saved in the output folder.----------------------------------------------------------------**The script will also print:**the selected input file paththe project directorythe output directorythe input data dimensions----------------------------------------------------------------
