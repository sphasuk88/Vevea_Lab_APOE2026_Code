# Vevea_Lab_APOE2026_Code
Code and demo datasets for reproducible analysis workflows

## Repository structure

This repository contains figure-specific and workflow-specific folders for reviewer access. Each folder includes:
- an `input` folder with demo datasets
- an `output` or `output_and_processing` folder for generated files
- an `R_code` or `R_script` folder with the analysis script
- a folder-specific `README.txt` file with run instructions

Current folders include:
- `representative_isoform`
- `IgG_subtraction`
- `fig3AB`
- `fig3D`
- `supfig2A`
- `supfig3B`

## How to use

1. Open the relevant folder in the repository.
2. Read that folder’s `README.txt` file.
3. Open the corresponding script in RStudio.
4. Run the script.
5. When prompted, choose the demo input file from that folder’s `input` directory.
6. Check the generated outputs in that folder’s `output` or `output_and_processing` directory.

## Notes

- These materials are provided for reviewer access during peer review.
- Demo datasets are included for workflow execution and code verification.
- Figure-specific and workflow-specific details, expected outputs, and input requirements are described in each folder’s `README.txt`.
- The demo datasets are intended to allow reviewers to run the code and verify the workflow, but may not reproduce every quantitative result from the full study dataset.
